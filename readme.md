# LÖVE test

## Intro
Exprimenting with Löve2D framework

https://love2d.org


## Setup

Fetch external module dependencies

```bash
git submodule init
git submodule update
```

## Run

```bash
love .
```

## 3D control

Using example from 3DreamEngine

- W, A, S, D: up, left, down, right
- lshift: down
- space: up
- mouse
